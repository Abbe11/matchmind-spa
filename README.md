# MatchMind ⚽

A Single Page Application (SPA) that lets users search for sports teams and view match data, standings, and fixtures.

## Features
- 🔍 Search for any sports team by name
- 📅 View last 5 results for a team
- 📆 View next 5 upcoming fixtures
- 🏆 View league standings
- ⚔️ Head-to-head stats between two teams

## Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- [TheSportsDB API](https://www.thesportsdb.com) (free public API)
- Fetch API for async data retrieval

## How to Use
1. Type a team name into the search bar
2. Press **Search** or hit **Enter**
3. Browse results, fixtures, and standings
